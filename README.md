## Start Application

1. yarn install
2. cd ios && pod install && cd ..
3. yarn ios

## You should have

- NodeJS version 18
- cocoapods 1.12
- yarn

## Technologies

React, React-Native, NodeJS, react-query, prettier, eslint

## Note

State management was realized through react-query but if needed it can be re-implemented by me on either Redux or RTK Query
